# BurgerHouse

Sitio web de BurgerHouse construido con Astro 5 y Tailwind CSS 4.

## Requisitos

- Node.js 20 o superior
- npm

## Scripts

- `npm run dev`: inicia el entorno de desarrollo
- `npm run build`: genera la version de produccion
- `npm run preview`: previsualiza la build local

## Stack

- Astro 5
- Tailwind CSS 4
- lucide-astro

## Estructura

```text
src/
  components/
  data/
  layouts/
  pages/
  styles/
public/
```
